# 📁 File Organiser (Node.js CLI)

A command-line **File Organiser application** built using **Node.js** that helps users organise files into folders based on file type.  
It also provides an option to **undo the organisation** and restore files back to their original location.

---

## ✨ Features

- Organise files into folders:
  - Image Files
  - Video Files
  - Document Files
  - Audio Files
  - Other Files
- Undo organise option
- Interactive command-line interface using Inquirer
- Option to manually enter a folder path or select system folders
- Progress bar while processing files
- Handles invalid paths safely

---

## 🛠️ Tech Stack

- Node.js
- fs/promises
- Inquirer
- Path & OS modules
